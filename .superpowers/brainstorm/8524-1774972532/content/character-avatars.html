Avatar Appears Throughout the Quest

Where the Avatar Shows Up
  • Top bar — avatar icon + class name + level
  • Question cards — small avatar reacts to answers
  • Block transitions — "The Warrior enters the Arena of Interests..."
  • Chart reveals — avatar celebrates with class-specific animation
  • Badge unlocks — class-flavored unlock message
  • Dashboard — avatar as profile picture
What Adapts per Class
  • Color palette — primary, accent, glow colors
  • Border style — sharp (warrior), rounded (sorceress), mixed (wanderer)
  • Transition text — class-flavored narration
  • Badge unlock text — "Battle Cry" vs "Spell Unlocked" vs "Discovery Made"
  • XP bar color — matches class accent
  • Background gradient — subtle tint shift
