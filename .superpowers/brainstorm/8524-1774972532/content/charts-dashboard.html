Key Design Decisions

RIASEC Hexagon
  • Recharts RadarChart component
  • Purple theme with translucent fill
  • Dots on vertices for clarity
  • Top types labeled below chart
  • Animates from center on first reveal
  • Morphs smoothly during confirmatory round
MBTI Spectrum Sliders
  • Custom component (not Recharts)
  • Dot position = score on -100 to +100
  • Purple dot = clear tendency
  • Gray dot + "Still emerging" = near center
  • Slides in from edges on first reveal
  • Dot glides smoothly during confirmatory
Locked Charts
  • MI bar chart + Values compass shown grayed out
  • 50% opacity with "Unlocks in Session 2" overlay
  • Creates anticipation without looking broken
  • Ghost data (gray placeholder bars/shapes)
Badge Row
  • Horizontal row at top of dashboard
  • Unlocked = purple circle with icon
  • Locked = gray circle with lock icon
  • Unlock animation: scale bounce + glow
  • 2 badges unlockable in Phase 1
